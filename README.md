#### **⚡ Get It Running Locally**

To run this locally:

1. Clone it:

   ```bash
   git clone https://github.com/shaaanuu/portfolio.git
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run it:
   ```bash
   npm run dev
   ```
